### Library Management System using Spring Boot


## How to run

- install java 21 & docker desktop via next commands
```cmd
    # windows 11 or 10 only.
    winget install EclipseAdoptium.Temurin.21.JDK
    winget install Docker.DockerDesktop
```
- open docker desktop
- run next command
```cmd
    ./gradlew bootRun 
```
